# Retail Business Intelligence Dashboard

## Project Overview
This project is an interactive Power BI dashboard built to analyze retail business performance.

## Tools Used
- Power BI
- Microsoft Excel

## Key Metrics
- Total Sales
- Total Profit
- Total Orders
- Total Customers

## Dashboard Features
- Sales by Category
- Profit by Category
- Sales Trend by Year
- Sales by Region
- Sales by Segment
- KPI Cards
  
## Business Insights
- Technology category generated the highest sales.
- West region contributed the highest revenue.
- Office Supplies generated strong profit.
- Sales increased over the years.
